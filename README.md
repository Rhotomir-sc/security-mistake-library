# Security Mistake Library

This repository documents mistakes, corrections, and lessons learned during my cybersecurity labs and study process.

I use this repository to keep track of what confused me, what I corrected, and what I learned from each mistake.  
The goal is to show realistic student-level growth, not perfect results.

---

## Purpose of This Repository

This repository answers one main question:

> How did I learn from my mistakes while studying cybersecurity?

This repository is designed to show:

- Lab mistakes and corrections
- Misunderstood concepts
- Debugging and troubleshooting notes
- Lessons learned from TryHackMe labs
- Reflections from Linux, networking, logs, and security monitoring practice
- A realistic learning process as a cybersecurity student
